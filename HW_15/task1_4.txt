1. Используя тестовую БД sqlite „chinook.db“ определите:
1) Сколько всего байт «занимают» песни из таблицы tracks
SELECT SUM(Bytes) FROM tracks

2) Сколько записей находится в таблицах employees и customers
SELECT SUM(result) FROM (
SELECT COUNT(EmployeeID) as result FROM employees
UNION
SELECT COUNT(CustomerID) as result FROM customers
)

3) Получить список треков tracks из альбома «A-Sides»
SELECT Name FROM tracks WHERE AlbumId = (SELECT AlbumId FROM albums WHERE Title = 'A-Sides')


4) Используя группировку (https://www.w3schools.com/sql/sql_groupby.asp) определите общую стоимость треков в каждом альбоме
SELECT AlbumId, Title as AlbumName, album_price as AlbumPrice
FROM albums
INNER JOIN ( 
SELECT AlbumId, SUM(UnitPrice) as album_price
FROM tracks
GROUP BY AlbumId
)
USING (AlbumId)
